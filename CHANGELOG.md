# Changelog

## 0.1.0

- Initial FunTaxFinder release.
- Added KO-based ASV screening from PICRUSt2 ASV-by-KO prediction tables.
- Added matched ASV abundance table extraction.
- Added optional taxonomy annotation and rank-specific splitting.
- Added optional FAPROTAX cross-validation mode.
- Added strict missing-KO handling to avoid silent false positives.
